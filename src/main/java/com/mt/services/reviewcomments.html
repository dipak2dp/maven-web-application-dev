Thomas Jockin and Bonnie Shaver-Troup, the font designers, have also uploaded the font files on Github under the Open Font License which allows you to use, modify and redistribute the fonts freely as long as they are not sold by themselves. To enable the Lexend font in your copy of Google Docs, open a new document inside Google Docs, click the Font dropdown in the toolbar and select More Fonts. Search for Lexend and select them to add them to your default font list.
